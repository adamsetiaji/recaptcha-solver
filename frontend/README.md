# reCAPTCHA Solver Monitor

This project is built using:
- Vite
- React
- TypeScript
- Tailwind CSS
- Recharts
- Lucide React Icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```
